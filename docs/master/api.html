Addon API Reference

Addon API has some useful API.

class AddonAPI
name: str

Addon Name

logger: Logger

Logger object for addon

appinfo: dict

Marutools runtime infomation

ext: str

Open file extention

ui: UI

UI controler

app: Core

Marutools Core

saved: bool

Flag for is saved file.

UI

UI controling class

class UI
parent

Parent UI object. If not having, it is None.

type: str
UI object type. UI object is showing Frame widget or window.
main: Main window
frame: Frame widget
dialog: Dialog widget(like Warning window…)
sub: Sub window
backend: str
Backend name.
tkinter:tkinter GUI library
classmethod changeTitle(title)

Change window title.

Warning

This works successfully only when UI object type is main or sub

Parameters

title (str) – Title to set.

classmethod changeIcon(icon_path)

Change window icon(it’s on title bar and task bar).

Warning

This works successfully only when UI object type is main or sub

Parameters

title (str) – Icon file path to set.

fullscreen(tf=None):

Change window to fullscreen.

Warning

This works successfully only when UI object type is main or sub

Parameters

tf (bool) – True to fullscreen. False to fullscreen. default(None) to nomal size.

changeSize(size):

Change window size.

Warning

This works successfully only when UI object type is main or sub

Parameters

size (str) – {width}x{height}+{x}+{y}

uisetting(frame, txt):

The widget for UI setting.

Parameters

  • frame (libtools.Frame) – Frame to show settings.

  • txt (Lang) – l10n text in dict like object

classmethod setcallback(name, callback)

Set callback.

Parameters

  • name (str) –

    Callback name.
    close: Fire on close window.
    macos_help: Click help menu button on Macos
    macos_settngs:Click settings menu button on Macos

  • callback (callable) – Callback function.

classmethod makeSubwindow(dialog=False, **options)

Make Subwindow

Parameters

dialog (bool) – Dialog mode.
